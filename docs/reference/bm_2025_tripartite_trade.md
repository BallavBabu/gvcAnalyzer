# BM_2025 tripartite GVC trade decomposition for one pair (s,r)

BM_2025 tripartite GVC trade decomposition for one pair (s,r)

## Usage

``` r
bm_2025_tripartite_trade(io, s, r)
```

## Arguments

- io:

  A `bm_io` object.

- s:

  Exporter country (name or index).

- r:

  Importer country (name or index).

## Value

Data frame for the pair (s,r).
