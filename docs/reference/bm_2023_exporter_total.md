# BM_2023 exporter-perspective decomposition of total exports of s

BM_2023 exporter-perspective decomposition of total exports of s

## Usage

``` r
bm_2023_exporter_total(io, s)
```

## Arguments

- io:

  A `bm_io` object.

- s:

  Exporter country (name or index).

## Value

A data frame with the exporter-total decomposition.
